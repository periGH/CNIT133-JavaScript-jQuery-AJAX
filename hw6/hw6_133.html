CNIT 133 Homework 6 - Objects

Part 1

Math method round can be used to round a floating-point number to an integer.

y = Math.round( x )

Math method sqrt can be used to find the square root of a number.

y = Math.sqrt( x );

Math method floor can be used to round a floating-point number to a specific decimal place.

y = Math.floor( x * 10 + .5 ) / 10;
y = Math.floor( x * 100 + .5 ) / 100;

Create a webpage that contains a script that has a function that uses Math.round, Math.sqrt and Math.floor to:

a) round the floating-point number to the nearest integer
b) calculate the square root of the floating-point number and round it to an integer
c) round the floating-point number to the nearest tenths position
d) round the floating-point number to the nearest hunderdths position
e) round the floating-point number to the nearest thousandths position

Your page should allow entering a floating-point number with at least 4 decimal positions. Your script should display the original floating-point number, the number rounded to the nearest integer, the square root of the number that is then rounded to an integer, the number rounded to the nearest tenth, the number rounded to the nearest hundredth, and the number rounded to the nearest thousandth. Be sure to use parseFloat() for the inputted number. Use a form input box for user input and a textarea to display the output. When displaying the output, be sure to use toFixed() for displaying the number to the nearest tenths, hundredths, and thousandths postions. Validate the input and display an error message if the input is invalid. It is acceptable to use jQuery for this problem.

Part 2

Create a webpage that contains a script using a form that allows the user to input several lines of text and a search character and uses the String method charAt to determine the number of occurrences of the search character within the text. Use a form textarea for the text and a form input box for the search character. If the search character is found within the text, display the number of occurrences of the search character in another form input box or a textarea. If the search character is not found within the text then display within a new window (using the window.open method), the message, 'Search character not found in text string!' along with the search character. Your new window should be a width of 300 pixels, height of 100 pixels, and should not obscure any of the main content already on the page.

Part 3

Create a webpage that does the following:

1. Using toString(), toLocaleString() and toUTCString() for the Date object, write the current date and time in these 3 formats. Using the getTimezoneOffset(), write the hours difference between PST or PDT (Pacific Standard Time or Pacific Daylight Time) and UTC, and the hours difference between EST or EDT (Eastern Standard Time or Eastern Daylight Time) and UTC. Note: EST is 3 hours ahead of PST (Pacific Standard Time). Note that for EDT (Eastern Daylight Time), the hours difference between EDT and UTC changes by one hour.

2. Input a telephone number as a string in the format (999) 999-9999. The script should use String method split to extract the area code as a token, the first 3 digits of the phone number as a token, and the last 4 digits of the phone number as a token. Display the area code in one text field and the seven-digit phone number in another text field. I suggest that you use the jQuery mask for input. Be sure to display the desired format for the phone number, e.g., (999) 999-9999.

Part 4

Write a script that inputs several lines of text in a form textarea and uses String method indexOf to determine the total number of occurrences of each letter of the alphabet in the text. Uppercase and lowercase letters should be counted together, e.g., 'A' and 'a' would count as 2 occurences of the same letter (use toLowerCase() or toUpperCase()). Store the totals for each letter in an array, and print the values in a vertical form textarea after the totals have been determined.
